# Bplace

A recreation of Reddit r/place

Done with [actix-web](actix.rs) and [React](https://reactjs.org/)

This was done hastily for fun, feel free to contribute, or use this as you like.

# Run-it

## With Docker

```
docker-compose up -d
```
Once started go to:
http://localhost:8080/

## With Cargo and Yarn

```
cargo run
# in another terminal
cd front;yarn start
```
